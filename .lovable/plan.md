

# Developer Portfolio for Indrajith A C

A modern, professional single-page portfolio website showcasing frontend development expertise with smooth animations, dark/light theme toggle, and recruiter-friendly design.

---

## 1. Hero Section
- Circular profile photo placeholder with a decorative border/glow effect
- Name, title ("Frontend Software Developer"), and a concise professional bio
- Three CTA buttons: **View Projects**, **Download Resume**, **Contact Me**
- Animated code-inspired background grid with subtle floating particles
- Smooth fade-in entrance animations

## 2. Theme Toggle
- Creative sun/moon animated toggle in the navigation bar
- Smooth color transitions across the entire UI
- Dark theme (deep neutrals) as default, with a clean light mode option

## 3. Navigation
- Sticky top navbar with smooth scroll to sections
- Mobile-responsive hamburger menu
- Active section highlighting as user scrolls

## 4. About Section
- Split layout: expanded bio text on one side, tech-themed visual/illustration on the other
- Career journey visual timeline component
- Education details (B.Tech, Sreepathy Institute, 2016–2020)

## 5. Skills & Expertise
- Skills grouped into categories: Frontend, Tools & Platforms, Cloud & Backend
- Interactive cards with tech icons and hover animations
- Clean developer-style grid layout

## 6. Experience Section
- Timeline/card layout for the role at EGuidance Labs LLP (Aug 2022 – Present)
- Expandable achievement bullets covering React apps, design systems, API integration, AWS/Firebase deployments, Agile workflow

## 7. Projects Showcase
- Visually rich project cards for **MemShots**, **Fjällräven Experience**, and **GoViral**
- Each card includes: description, key contributions, tech stack chips, and placeholder links for GitHub/demo
- Smooth hover animations and section reveal effects

## 8. Contact Section
- Email, phone, location displayed cleanly
- LinkedIn and GitHub icon links
- Simple contact form (name, email, message) with validation
- Subtle gradient/glow accents

## 9. Design & Polish
- Responsive-first layout (mobile → desktop)
- Smooth scroll-triggered section reveal animations
- Micro-interactions on buttons, cards, and links
- Professional typography with strong visual hierarchy
- Performance-optimized with clean component architecture
